DECENTRALIZED & ANONYMOUS
All communication is through the user location hiding Tor network. The Bitmessage network is used for posting and buyer/seller communication. Bitcoin is used for payments.
TWO PARTY ESCROW
Buyer and seller lock mutual security deposits and payment into a Bitcoin transaction that can only be released when both agree on payment or refund. No escrow agent or trust network is needed.
FREE & OPEN SOURCE
Free client, no fees. Open source code available on Github under an MIT license.
